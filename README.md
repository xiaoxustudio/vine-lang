# Vine Lang

Vine 是一种旨在便于使用和学习的解释型编程语言。它是一种简单的语言，基于 TypeScript 构建，但具有更现代的语法和功能

# 构建

项目使用 pnpm 作为包管理器

```bash
pnpm install
```

## dev

```bash
pnpm run dev
```
