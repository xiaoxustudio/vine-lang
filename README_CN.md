<h2 align="center">Vine Language</h2>

- [简体中文](./README_CN.md)
- [English](./README.md)

`Vine` (藤蔓) 是一种旨在便于使用和学习的解释型编程语言。它是一种简单的语言，基于 TypeScript 构建，但具有更现代的语法和功能。

# 开发

项目使用 pnpm 作为包管理器

```bash
pnpm install
```

## dev

本地运行

```bash
pnpm run dev
```

## build

构建 exe

```bash
pnpm run build
```

## repl

Vine 提供了 repl 功能，可以在命令行中直接运行代码

```bash
pnpm run repl

Vine > print("Hello, World!")
```

## interpret

同样也可以运行后缀名为`.vine`的代码文件

```bash
pnpm ipt <path>
```
