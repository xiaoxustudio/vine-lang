# Path (vine:path)

Utility functions for path operations

## file

```vine
file()
```

Get the current file path

## dir

```vine
dir()
```

Get the current directory path

## extname

```vine
extname(...args: string)
```

Get file extension

## resolve

```vine
resolve(...args: string)
```

Resolve path

## join

```vine
join(...args: string)
```

Join paths

## isAbsolute

```vine
isAbsolute(...args: string)
```

Check if path is absolute
