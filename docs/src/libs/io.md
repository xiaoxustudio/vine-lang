# 文件 (vine:io)

关于文件的操作

## 判断文件或文件夹是否存在

```vine
exists(pathName:string)
```

## 读取文件内容

```vine
read(pathName:string)
```

## 写入文件内容

```vine
write(pathName:string, content:string)
```

## 删除文件或文件夹

```vine
delete(pathName:string)
```

## 创建文件夹

```vine
mkdir(pathName:string)
```

## 删除文件夹

```vine
deleteDir(pathName:string)
```

## 获取文件或文件夹信息

```vine
info(pathName:string)
```
