# 路径 (vine:path)

关于路径操作的工具函数

## file

```vine
file()
```

获取当前文件路径

## dir

```vine
dir()
```

获取当前目录路径

## extname

```vine
extname(...args: string)
```

获取文件扩展名

## resolve

```vine
resolve(...args: string)
```

解析路径

## join

```vine
join(...args: string)
```

拼接路径

## isAbsolute

```vine
isAbsolute(...args: string)
```

判断路径是否绝对路径
